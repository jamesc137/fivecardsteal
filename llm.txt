## Project Overview
Five Card Steal is a strategic card game that combines memory, theft mechanics, and poker hand rankings. The game is designed for 3-5 players and takes approximately 15-25 minutes to play. It's suitable for players aged 12 and up.

## Game Description
Five Card Steal is a unique card game that merges traditional poker mechanics with strategic card stealing and memory elements. Players must build the best 5-card poker hand while navigating a dynamic gameplay system that allows them to steal cards from opponents, draw from a deck, or utilize discarded cards.

## Key Game Features
- **Strategic Depth**: Combines memory, timing, and tactical decision-making
- **Card Stealing Mechanics**: Players can steal cards from opponents with restrictions
- **Poker Hand Rankings**: Uses standard poker hand evaluation
- **Memory Challenge**: Optional hard mode that prevents looking at cards after setup
- **Quick Play**: 15-25 minute game duration
- **Family Friendly**: Suitable for ages 12+

## Game Components
- 1 Standard 52-card deck (no jokers required)
- Optional: Player mats with 5 card slots and viewing mirrors

## Gameplay Mechanics
1. **Setup**: Each player receives 5 cards face down, arranged left to right
2. **Turn Actions**: Players can draw from deck, steal from opponents, or use discard row
3. **Card Stealing**: Players can steal cards by position, with anti-steal-back rules
4. **Game End**: Triggered when 5th card is added to discard row
5. **Victory**: Highest-ranking poker hand wins
